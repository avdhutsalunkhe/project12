# Data Schema Documentation

The system relies on a central source of truth for all SHL assessments, located at `data/catalog.json`.

This file is automatically generated by the web scraper (`scraper/scraper.py`) which paginates through the public SHL product catalog.

## `catalog.json` Structure

The root of the document contains an object with two fields:
- `total_assessments`: Integer count of scraped items.
- `assessments`: Array of Assessment objects.

### Assessment Object

Each assessment in the `assessments` array has the following structure:

```json
{
  "name": "Verify - Numerical Reasoning",
  "url": "https://www.shl.com/solutions/products/product-catalog/view/verify-numerical-reasoning/",
  "catalog_type": "individual",
  "description": "Measures the ability to make correct decisions or inferences from numerical or statistical data.",
  "test_type_code": "A",
  "test_type": "Ability & Aptitude",
  "duration_minutes": 17,
  "remote_testing": true,
  "job_levels": [
    "Graduate",
    "Mid-Professional"
  ],
  "languages": [
    "English - US",
    "French",
    "Spanish - Modern"
  ],
  "fact_sheet_urls": [
    "https://www.shl.com/assets/fact-sheets/verify-numerical.pdf"
  ],
  "scrape_error": false
}
```

### Field Definitions

| Field | Type | Description |
|---|---|---|
| `name` | string | The canonical name of the assessment. |
| `url` | string | The direct link to the SHL catalog detail page. |
| `catalog_type` | string | Usually `"individual"` to denote a single test solution. |
| `description` | string | Full text description extracted from the page. Used heavily for semantic vector matching. |
| `test_type_code` | string | Single character code (e.g., `A` for Ability, `P` for Personality). |
| `test_type` | string | Human-readable category of the test. |
| `duration_minutes` | integer \| null | Extracted testing time. Important for duration constraints. |
| `remote_testing` | boolean \| null | Indicates if the test supports unproctored/remote administration. |
| `job_levels` | array[string] | Target seniority levels (e.g., "Manager", "Graduate"). Used by the Hybrid Ranker. |
| `languages` | array[string] | Available language translations. |
| `fact_sheet_urls` | array[string] | Links to PDF fact sheets, if available. |
| `scrape_error` | boolean | True if the scraper failed to parse the detail page. |

## Hallucination Prevention
To enforce a zero-hallucination policy, the `HallucinationGuard` class loads this JSON file into memory on startup. It builds a fast lookup `Set` of all valid names and URLs. Any recommendation generated by the system that does not perfectly match a name/URL in this schema is rejected before returning a response to the user.
